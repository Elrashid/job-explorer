## Role Category
This is an IT role focused on designing and maintaining institutional data architecture at a university. It involves database design, Azure cloud platform management, data pipeline development, and cross-functional collaboration.

## Required Skills
### Technical
- Database design and optimization
- Azure cloud platform (Data Factory, Data Lake, Data Warehouse, Data Lakehouse)
- Conceptual, logical, and physical data modeling
- Data pipeline development and ETL processes
- System integration across platforms
- Security and backup procedures
- Legacy data migration

### Soft Skills
- Cross-functional collaboration with university departments
- Analytical thinking and troubleshooting
- Communication with diverse stakeholders

## Qualifications
- **Education:** Bachelor's degree (with 4 years experience) or Master's degree (with 2 years experience), plus supervisory experience
- **Experience:** 2-4 years depending on degree level
- **Certifications:** Microsoft Azure Solutions Architect, Azure Data Engineer Associate (required)

## Key Responsibilities
1. Develop database solutions for efficient and secure information management
2. Improve system performance through testing, troubleshooting, and integration
3. Install and configure information systems across platforms
4. Analyze structural requirements for new software and applications
5. Transfer data from legacy systems to new solutions
6. Design conceptual and logical data models and flowcharts
7. Optimize new and existing database systems for speed and reliability
8. Define security and backup procedures
9. Create and maintain end-to-end data platform architectures on Azure
10. Develop data pipelines using Azure Data Factory
11. Collaborate with university functions to identify future data needs

- Azure Data Warehouse
